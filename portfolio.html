1.1.9 Project:

"Luca Clicker is a idle style game that uses loops and iteration to create automation with management.
When The player clicks the cookie, it fires off an event that updates the cookie variable, then the scoreboard to that new variable.
An infinite while loop allows the cookie to spin in a circle for eternity, checks to see if any auto-clickers have been bought during the end of it's current cycle.
Iteration is used to determine if the player is able to afford each type of autoclicker using the variables that count the cookies.
All of the buttons and the cookie are indidual turtles that use to onclick event to detect keystrokes made hovering over each image."

1.2.5 Project:

"Protect the Kingdom from Bugs is a castle defense style hame where your cursor is your main weapon. Varieties of bugs
attempt to invade your kingdom using turtle and random methods inside fuctions which are nested in two main threaded loops.
We use iteration to determine whether the bugs were successful at invading the kingdom and if they do, a global score variable is altered.
We enhance the effects of the game by using pygame's mixer to add sound effects as well as backround music. Finally, we had a seperate
script in order to handle our leaderboard's function, retriving it's data from a
txt file that stores names and scores."

Scratch Project

"Flappy Food is a Flappy-Bird style game that I created in scratch for my third project. The game uses player input from the keyboard in order to move the food on the screen.
Using procedures, obstacles are spawned at intervals in order to create a challenge for the player. When the player dies in the game (ex: hitting a toilet), the game will end and the players
current score and highest score of the session is displayed. Sound effects were also added in order to enhance the players experience for when the food player flaps, dies, and is awarded a point."
